Key moments in the history of the Web

The Web developed step by step: first as a document-sharing system, then as a public communication space.

1960s–1970s: Networks before the Web

Before the Web, computer networks already existed. The basic idea was to connect computers so they could exchange information. This prepared the ground for later technologies such as TCP/IP.

1980s: TCP/IP becomes important

TCP/IP is a group of communication protocols. TCP helps divide and reassemble data packets. IP helps identify where data has to go using IP addresses.

1989–1991: Tim Berners-Lee and the World Wide Web

Tim Berners-Lee proposed a system based on hypertext, URLs, HTTP, and HTML. This made it possible to connect documents through links and access them with a browser.

1990s: Browsers and public access

Web browsers made the Web easier to use. People no longer needed to understand complex commands: they could click links, read pages, and navigate information visually.

2000s–today: Dynamic and social Web

The Web became more interactive. Users started to create content, share media, use platforms, and access websites from many devices, especially smartphones.
