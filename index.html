A simple educational resource for humanities students

From connected computers to a global system of meaning

This website explains the history of the Web in a simple way. It connects historical moments with basic technical concepts such as client-server architecture, IP addresses, DNS, TCP/IP, HTML tags, CSS properties, and static websites.

Start the timeline

1. Historical development

The Web was not born as a social network or a shopping platform. It began as a way to share documents through hyperlinks.

2. Technical structure

Behind every website there is communication between a browser, a server, and protocols that allow data to travel correctly.

3. Semiotic relevance

The Web is also a cultural environment: pages, links, menus, images, and interfaces organize how people access information.

Why this topic matters

For humanities students, the history of the Web is important because it shows how a technical invention became a space for communication, memory, identity, and cultural production. The Web is not only a technology: it is also a system that gives structure to information and meaning.
