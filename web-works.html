How the Web works

This page explains the technical process behind opening a website in a browser.

Client-server architecture

The client is the user's device and browser. The server is the computer that stores the website files. When we type a website address, the browser sends a request and the server sends back a response.

DNS, IP addresses, and TCP/IP

A domain name such as example.com is easy for humans to remember, but computers use IP addresses. DNS works like an address book: it translates the domain name into the correct IP address. Then TCP/IP helps the data travel across the network in packets.

HTML and CSS

HTML gives structure to a page. For example, tags such as <header>, <nav>, <section>, and <footer> divide the content into meaningful parts. CSS controls the visual style through properties such as color, font-family, margin, padding, border-radius, and background-color.
